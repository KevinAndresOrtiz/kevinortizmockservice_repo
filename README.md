## Instalacion
  ### Pre Instalacion
  Se debe tener instalado  Nestjs en la maquina donde se ejecuta el proyecto, el comando para la instalacion es el siguiente
  ```bash
  npm i -g @nestjs/cli
  ```
Una vez descargado el proyecto , ejecuta el siguiente comando
```bash
$ npm install
```
